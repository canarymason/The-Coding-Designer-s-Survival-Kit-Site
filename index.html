Teasers

A Nice, Clean Place to Start

Start your project with sensible defaults that help you get straight to the good stuff.

Step-By-Step Instructions

This isn’t old-school css. Get your hands dirty with all the new Sass and Compass tricks. Let The Kit be your guide.

Color Tools

Take Sass color variable a step further. The Kit has all the named css colors as sass variables, and even a system for generating color schemes.

Easy Webfont Integration

The Kit establishes a system for incorporating fonts from services like Typekit, or your own font files.

Sass Mixins

Save precious key-strokes and make your css DRY. The Kit includes Sass mixins for creating better links, spacing elements, and even creating schmancy “page curl” shadows.

HTML5, Of Course

Anyone can style a div. Let’s dive deeper. The Kit includes all the examples from developers.whatwg.org for you to style like the go-getter you are.

Common UI Patterns

The Kit lays out common elements you’re likely to style in any real site. All the hits like menus, pagers, lists, breadcrumbs are here.

Drupal UI Patterns

The Kit also includes all the common patterns found in Drupal 7. Tabs, logins, nodes, blocks, and views? Yeah.

Code Viewer

As you’re designing in the browser, it’s nice to see the markup your working with. The Kit opens a window and lets you see what you need to.
